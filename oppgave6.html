Git og Github

Github er en webbasert Git som igjen er en form for versjonskontroll av kodefiler. Poenget med Git i seg selv er å holde styr på hvilke versjoner du har opprettet av en fil i tilfelle du vil gå tilbake til tidligere endringer. Git ble grunnlagt av Linus Torvald i 2005; samme person som skapte de første versjonene av Linux i 1991. GitHub fungerer som en samleside for alle beholderne du lager med Git og ble grunnlagt i 2008 av Tom Preston-Werner, Chris Wanstrath og PJ Hyett.
